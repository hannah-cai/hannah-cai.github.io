p5.js

p5.js is a Processing-based javascript library. It's taught in the class "Introduction to Computing for Creative Practice" (15104); I signed up for the class on impulse and ended up really enjoying it. 15104 was my first introduction to coding, so a lot of my code here is pretty messy (and hard-coded). Hopefully, you'll still be able to see improvement as the projects advance.

For each of these projects, I generally worked in Illustrator first, then imported the data to p5 and added interaction/code. Documentation is linked below each project; some of the projects look a bit different in the documentation, since I've gone back and refined some since then.
